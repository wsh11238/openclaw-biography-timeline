# 仓库简介 / Repository Description

人物时间线生成器：一套把人物公开资料整理成结构化年表、关键线索、成就争议、关系网络、深度复盘和资料来源的工具包。适合人物志、创业者传记、历史研究、课程资料和长篇内容知识库。

Biography timeline generator: a toolkit for turning public biographical research into structured timelines, key facts, achievements and controversies, relationship maps, long-form review sections, and source notes.
