## V2EX / 分享创造

标题：

```text
开源了一个可直接 Fork 的人物时间线生成器，真实中文界面 + Demo GIF + 中英文 README
```

正文：

```text
最近把一个真实项目清理成了公开仓库，去掉了密钥、私有域名和部署痕迹，只保留可复用的前端结构、真实中文截图和 Demo GIF。

它主要解决的问题是：把零散公开资料整理成时间轴、关系网络、深度复盘和来源说明。

适合这些场景：
- 创业者传记
- 人物志网站
- 历史研究
- 课程资料
- 长篇内容知识库

Fork 后主要替换 人物 JSON 和来源文件，就可以改成自己的项目。

GitHub: https://github.com/wsh11238/openclaw-biography-timeline
```
